Aplicación con Supabase

Esta aplicación fue diseñada para un comercio local, con una base de datos Supabase conectada en tiempo real.

Utilicé Electron para que funcionara como una app de escritorio accesible y portable.

Fue mi más reciente proyecto que elaboré en poco tiempo y su utilidad en el negocio es para registrar los movimiento de entrada y salida de productos. Almacenar las boletas y descargarlas en archivos pdf todas las ventas asignadas a un empleado.

Desarrollé tanto el backend para la petición de información en supabase, la actualización y modificación de datos, como el front para que sea utilizado por el usuario, el empleador y agregar productos cargando el código de barras.

Utilizé vite, node.js.

Características técnicas

  • CRUD completo de productos
  • Autenticación de usuarios
  • Distribución multiplataforma con Electron
